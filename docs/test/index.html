Installation Instructions
  1. Download the APK from the link above (or from the shared Google Drive folder).
  2. On your Android device, go to Settings → Security (or search for "Install unknown apps").
  3. Allow your browser to install apps from unknown sources.
  4. Open the downloaded APK and install it.
This is normal for sideloading test builds. You can disable the setting again after installing.
