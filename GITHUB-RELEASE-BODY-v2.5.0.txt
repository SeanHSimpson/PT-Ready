**PT Ready v2.5.0 – Early Access Test Release**

This is the first release being shared with external testers.

### Key Improvements in This Version
- Complete redesign of the UFPM Member Profile view (much more compact and professional)
- Prominent color-coded score display in the header
- New workflow cards for "Test" and "AI Plan" with relevant member context
- Inline history mini-chart + summary statistics
- Enhanced AI training plan generation, including pregnancy/postpartum mode
- General improvements to roster management and navigation
